    Panduan Settingan

    Settingan yang wajib anda ubah ketika menggunakan script awal ini adalah

    • Settingan Database

    • Settingan Email

    • Settingan URL

    • Settingan API

    Database

    Database kita menggunakan database mysql, jadi anda harus mengubahnya di file .env pada folder backend, file yang wajib diubah adalah Dokumentasi lebih lengkap bisa check di Codeigniter 4 Database

    database.default.hostname = localhost

database.default.database = // nama database anda

database.default.username = //username akun yang terhubung dengan database anda

database.default.password = //password akun yang terhubung dengan database anda

database.default.DBDriver = MySQLi

database.default.DBPrefix =

database.default.port = 3306

    Email

    pada contoh ini saya gunakan phpmailer untuk mengirim email, jadi anda harus mengubahnya di file .env pada folder backend, file yang wajib diubah adalah pada baris ini anda harus mengubahnya sesuai dengan email anda, dan pastikan anda mengaktifkan akses kurang aman pada email anda, untuk lebih lengkapnya bisa check di Codeigniter 4 Email, untuk setting password client jika mau pakai Google Mail Setting email gmail

    app.email = xxx@gmail.com
app.email.pass = xxxx
app.email.host = smtp.gmail.com
app.email.port = 587
app.email.encryption = tls

    URL

    untuk mengubah url anda bisa mengubahnya di file .env pada folder backend, file yang wajib diubah adalah

    app.baseURL = 'http://localhost:8080/'

    API

    Untuk membuat sebuah API agar jalan memerlukan beberapa persyaratan yaitu : membuat sebuah controller, model, dan setting routes, untuk lebih lengkapnya bisa check di Codeigniter 4 API

    untuk membuat controller, model, dan isi database api di codeigniter 4 anda bisa membuatnya dengan cara buka terminal pada folder backend dan ketikkan perintah berikut

    • Tutorial membuat Controller Restful Codeigniter 4

    • Tutorial membuat Model Codeigniter 4

    • Tutorial membuat Table di dalam Database Codeigniter 4

    atau bisa juga dengan cara dibawah ini

    //buat controller
php spark make:controller sebuahController  --restful

//buat model
php spark make:model sebuahModel

//buat database
php spark make:migration nama_table_didatabase
